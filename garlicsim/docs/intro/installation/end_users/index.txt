========================================
End-user GarlicSim installer for Windows
========================================

Use this installer you just want to run GarlicSim as a desktop app, like
Photoshop or Word. You don't have to know programming to use GarlicSim this
way. (Currently Windows only.)

.. image:: /images/windows_installer.png
    :align: left
    :target: https://sourceforge.net/projects/garlicsim/files/GarlicSim-0.6.3.exe/download

`Download end-user Windows GarlicSim 0.6.3 installer
<https://sourceforge.net/projects/garlicsim/files/GarlicSim-0.6.3.exe/download>`_

This installer will give you :mod:`garlicsim`, :mod:`garlicsim_lib` and
:mod:`garlicsim_wx`.
